How to Create an Extension

  1. Create a .hsx file with your logic.
  2. Create a .js extension file for your .hsx code.
  3. Create a .html file to bring them together in the browser.
  4. Create test.txt and success.txt for submission verification.
  5. Upload your files below to test and submit your extension!
